# 🌙 Wake up, Little Girl!

**Wake up, Little Girl!** nasceu como um projeto pessoal, feito com carinho e dedicação para homenagear uma pessoa especial.

A personagem **Agda**, os cenários, os diálogos e os momentos da história foram construídos com a intenção de representar sentimentos reais através de uma narrativa fantástica. A ideia é transformar lembranças, afeto, cuidado e conexão em algo jogável, como se cada fase fosse uma parte de uma memória ou de um sentimento.

O jogo carrega uma proposta íntima: mostrar que algumas pessoas marcam tanto nossa vida que acabam se tornando parte dos nossos sonhos, dos nossos pensamentos e da nossa forma de enxergar o mundo.

Por isso, cada detalhe foi criado para transmitir uma sensação de carinho, mistério e descoberta.

---

## 🎮 Como jogar

🌐 **Jogue online:**
👉 [Clique aqui para jogar](https://xk4du.github.io/wake-up-little-girl/)

💻 **Rodar localmente:**

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Clique em **Jogar**.
4. Use as teclas para movimentar a personagem.
5. Interaja com objetos, personagens e portais para avançar na história.

### Controles

| Tecla      | Ação                                         |
| ---------- | -------------------------------------------- |
| `W`        | Andar para cima                              |
| `A`        | Andar para esquerda                          |
| `S`        | Andar para baixo                             |
| `D`        | Andar para direita                           |
| `E`        | Interagir com objetos, personagens e portais |

## ✨ Principais recursos

* 🕹️ Movimentação 2D com teclado.
* 💬 Sistema de diálogos narrativos.
* 📱 Conversas por celular dentro do jogo.
* 💌 Cartas e objetos interativos.
* 🧩 Quebra-cabeça.
* 🏆 Sistema de conquistas.
* 🎵 Música em cenas específicas.
* 🌌 Cenários cósmicos, românticos e misteriosos.
* 🌀 Portais para transição entre fases.
* 🚧 Sistema de colisão com móveis, objetos e limites do mapa.
* 🎞️ Transições suaves entre cenas.
* 🖼️ Assets personalizados em estilo pixel art.
* ❤️ Narrativa emocional.
